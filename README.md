##001 数字にする
ユーザーが入力した文字列を数字に変換して、その数字を表示する。ただし、ユーザーが入力した文字列が数字でない場合は、"数字ではありません"と表示する。

- case1:
  - 入力: １2
  - 出力：１2

• case2:
• 入力: おっはよう
• 出力: "数字ではありません"
